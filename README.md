# Rock Paper Scissors Game Using Python

This project involves the development of a simple command-line based Rock Paper Scissors game using the Python programming language. The objective of this project is to demonstrate fundamental programming concepts such as user input handling, conditional logic, and the use of built-in libraries. The game is designed to allow a single user to play against the computer in an interactive manner through the terminal.

In this application, the computer randomly selects one of the three possible moves: rock, paper, or scissors, using Python’s built-in random module. After the computer has selected its move, the user is prompted to enter their choice. The program then compares the user’s input with the computer’s selection based on the standard rules of the Rock Paper Scissors game to determine the result. The outcome is displayed as a win, loss, or tie.

The project is implemented entirely in Python without the use of any external libraries or frameworks, making it lightweight and easy to execute on any system with Python installed. The logic of the game is implemented using if-else conditional statements, which helps beginners understand decision-making structures in programming.

To execute the program, Python must be installed on the system. The user can run the application by navigating to the project directory and executing the file using the command `python rps_ai.py`. The program runs in the terminal and provides immediate feedback based on the user’s input.

This project serves as an introductory example for beginners learning Python and can be further enhanced by adding features such as score tracking, multiple rounds of gameplay, improved input validation, or a graphical user interface. It is suitable for academic submissions, internship tasks, and basic portfolio projects.
